# 兼职管理系统1

#### 介绍
兼职管理系统拥有两种角色，分别为管理员和用户。

管理员：用户管理、学校管理、兼职管理、订单管理

用户：发布兼职、报名兼职、个人信息查看（查看发布的信息，收到的报名信息等）
